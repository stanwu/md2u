課題

心当たりありませんか？

既存の Markdown ツールは macOS 専用・重すぎ・見た目が悪い・クローズドソースのどれか

Windows の問題

メモ帳の Markdown レンダリングが残念すぎる

Windows 11 のメモ帳は Markdown に対応しているものの、レンダリング品質が低く、フォント・余白・コードブロックが見苦しい。ドキュメントのレビューには使えない。

macOS 専用

Marked2 は最高だが macOS だけ

Marked2 は macOS で圧倒的な人気を誇る Markdown プレビューアーで、テーマも豊富でエクスポートも強力。しかし Windows や Linux では使えない。

🐘
重すぎ

他のツールは大げさすぎる

VS Code・Typora・Obsidian は高機能だが起動が遅くメモリを食う。.md ファイルをさっと確認したいだけなのに、エディタ全体を起動する必要はない。

🔒
クローズドソース

クローズドで機能制限・有料

多くのクロスプラットフォーム Markdown ツールはクローズドソースで、機能制限・更新が遅い・サブスクが必要。問題が起きても自分で直せない。

体験の差

MD を開くのは PDF より難しくあるべきでない

PDF をダブルクリックするときれいな文書が開く。.md をダブルクリックすると # や ** や ``` の記号が羅列される。同じ文書なのに扱いが全然違う。

だから自分で作った ✦

md2u — Tauri で構築したクロスプラットフォーム Markdown プレビューアー。
Obsidian と同じレンダリングエンジンをデフォルト搭載し、軽量・オープンソース。
Windows / Linux でも同じ体験を。もう妥協しなくていい。

機能

必要なものが揃ってる

無駄なし。Markdown プレビューに特化。

⚡

即座にプレビュー

.md ファイルをウィンドウにドラッグ、または md2u file.md で即レンダリング。保存後に自動更新。

🎨

10 種類の CSS テーマ

Obsidian、GitHub、Swiss、Ink、Multi-Column、Grump などを内蔵。Custom CSS 対応。⌘1–⌘9 で切替。

🔍

全文検索

Ctrl+F / Cmd+F で Find Bar を開き、マーカーでリアルタイム強調表示。Whole words・大文字小文字・Regex 対応。

📊

Mermaid 図表

フローチャート・シーケンス図・ガントチャートをインラインレンダリング。エクスポート時も SVG を保持。

📤

ワンクリックエクスポート

HTML（CDN 付き）・PDF（A4 ページ区切り）・DOCX・MD・OPML をアイコンクリックでエクスポート。

🔓

完全オープンソース・永久無料

GNU GPL v3 採用。商用・個人問わず誰でも無料で利用可能。ソースは GitHub 公開、PR 歓迎。
