痛点

你一定有过这些困扰

市面上的 Markdown 工具，要么只有 macOS 能用，要么太重、太丑、闭源

Windows 痛点

记事本渲染结果惨不忍睹

Windows 11 内置记事本虽然支持 Markdown，但渲染质量粗糙，字体、间距、代码块都差强人意，用来 review 文档简直是折磨。

macOS 独占

Marked2 太好用，却只有 macOS

Marked2 是 macOS 上公认最优秀的 Markdown 预览器，主题丰富、导出强大。但换到 Windows 或 Linux 就完全失去这个体验。

🐘
太重

其他方案太臃肿

VS Code、Typora、Obsidian 功能强大但启动慢、内存占用高。你只是想快速预览一个 .md 文件，不需要一个完整的编辑器生态。

🔒
闭源

闭源软件，功能有限还要钱

许多跨平台 Markdown 工具是闭源的，功能受限、更新慢，甚至要订阅付费。出了问题无法自己修，只能等官方响应。

体验落差

看 MD 不应该比看 PDF 难

双击 PDF，画面清晰干净，字体、排版、图表一目了然。双击 .md，却只看到一堆 #、**、``` 符号。同样是文档，待遇却天差地别。

所以我就直接做了一个 ✦

md2u — 基于 Tauri 打造的跨平台 Markdown 预览器，
默认采用与 Obsidian 相同的渲染引擎，轻量开源，Windows / Linux 都能用，
终于不用在 macOS 和其他平台之间做出妥协。

功能

你需要的，全都有

不多余、不臃肿，专注在 Markdown 预览这件事上做到最好

⚡

即开即用，秒速预览

拖拽 .md 文件到窗口，或命令行 md2u file.md，立即渲染，保存后自动更新。

🎨

10 种 CSS 主题

内置 Obsidian、GitHub、Swiss、Ink、Multi-Column、Grump 等主题，支持 Custom CSS 自定义，⌘1–⌘9 快速切换。

🔍

全文搜索

Ctrl+F / Cmd+F 打开 Find Bar，荧光笔实时标注，支持 Whole words、Case sensitive、Regex。

📊

Mermaid 图表

直接在 Markdown 中渲染流程图、时序图、甘特图等，export 时 SVG 完整保留。

📤

一键导出

支持 HTML（含 CDN）、PDF（A4 分页）、DOCX、MD、OPML，点图标即可导出。

🔓

完全开源，永久免费

采用 GPL v3 授权，任何人均可免费使用（含商业用途），源码在 GitHub 公开，欢迎 PR。
